SearchFor¶

Perform a search for a text string on a page.

This method is identical to class method Page.searchFor().

SearchFor(page, text, hit_max = 16)¶
Parameters:
  • page (Page) – A page created by Document.loadPage().
  • text (string) – A string to be searched for.
  • hit_max (int) – The maximum number of occurrences to be searched for.
Return type:

list
Returns:

A Python list of Rect rectangles of which each surrounds one occurrence of the string.
