.. _Device:

================
**Device**
================

The different format handlers (pdf, xps, etc.) interpret pages to a "device". These devices are the basis for everything that be done with a page, like rendering, text extraction and searching. What will actually be done with a page, depends on the argument type used in constructing a device.

Class API
==========

.. class:: Device

   .. method:: __init__(self, object)

      Constructor for either a pixel map or a display list device.

   .. attribute:: object

      An object representing one of :ref:`Pixmap`, or :ref:`DisplayList`

      :type: instance

   .. method:: __init__(self, textsheet, textpage)

      Constructor for a text page device.

   .. attribute:: textsheet

      A :ref:`TextSheet` object.

      :type: instance

   .. attribute:: textpage

      A :ref:`TextPage` object.

      :type: instance
