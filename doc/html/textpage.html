TextPage¶

TextPage contains all the text in a page. Create a new TextPage by Context.new_text_page().

Methods¶

get_text_len() Get the length of the text.
new_text_device() Create a device to extract the text on a page.
print_text_page() Output a page to a file in UTF-8 format.
print_text_page_html() Output a page to a file in HTML format.
print_text_page_xml() Output a page to a file in XML format.
search() Search string in the current page.
text_char_at() Get the TextChar at the given position.

new_text_device(sheet)¶

Create a device to extract the text on a page.

Gather and sort the text on a page into spans of uniform style, arranged into lines and blocks by reading order. The reading order is determined by various heuristics, so may not be accurate.

Parameters:sheet (TextSheet) – The text sheet to which styles should be added. This can either be a newly created (empty) text sheet by Context.new_text_sheet(), or one containing styles from a previous text device. The same sheet cannot be used in multiple threads simultaneously.
Return type:Device
