Pixmap¶

Pixmaps represent a set of pixels for a 2 dimensional region of a plane. Each pixel has n components per pixel, the last of which is always alpha. The data is in premultiplied alpha when rendering, but non-premultiplied for colorspace conversions and rescaling.

Create a new pixmap by Context.new_pixmap(), Context.new_pixmap_with_bbox(), Context.new_pixmap_with_data(), or Context.new_pixmap_with_bbox_and_data().

Class API¶

class Pixmap¶
clear_pixmap(self, value=0)¶

Clears a pixmap.

Parameters:value (int) – Values in the range 0 to 255 are valid. Each component sample for each pixel in the pixmap will be set to this value, while alpha will always be set to 255 (non-transparent). If not given, all components of all pixels in a pixmap will be set to 0.
gamma_pixmap(self, gamma)¶

Apply gamma correction to a pixmap. All components of all pixels are modified (except alpha, which is unchanged).

Parameters:gamma (float) – The gamma value to apply; 1.0 for no change.
get_bbox(self)¶

Get a bounding box for a pixmap.

Return type:BBox
get_samples(self)¶

Get the components data for all pixels. Samples is a block of memory width * height * n bytes of memory in which the components are stored. The first n bytes are components 0 to n-1 for the pixel at (x,y). Each successive n bytes gives another pixel in scanline order. Subsequent scanlines follow on with no padding.

Return type:string
get_width(self)¶

Get the width of the region in pixels.

Return type:int
get_height(self)¶

Get the height of the region in pixels.

Return type:int
get_colorspace(self)¶

Get the colorspace the pixmap is in.

Return type:ColorSpace
get_components(self)¶

Get the number of color components n in the image. Always includes a separate alpha channel. For mask images n=1, for greyscale (plus alpha) images n=2, for rgb (plus alpha) images n=3.

Return type:int
invert_pixmap(self, bbox)¶

Invert all the pixels in the pixmap(or a given bounding box). All components are inverted(except alpha, which is unchanged).

Parameters:bbox (BBox) – Invert all the pixels in the bbox. If not given, the whole pixmap will be inverted.
new_draw_device(self, bbox)¶

Create a device to draw on a pixmap. The pixmap is not cleared by the draw device, see clear_pixmap() for how to clear it prior to calling new_draw_device().

Parameters:bbox (BBox) – bounding box to restrict any marking operations of the draw device. If not given, the draw device will be created on the whole pixmap.
Return type:Device

Convert a pixmap from premultiplied to non-premultiplied format.

write_pam(self, filename, savealpha)¶

Save a pixmap as a pam.

Parameters:
  • filename (string) – The filename to save as (including extension).
  • savealpha (int) – Save alpha or not.
write_pixmap(self, filename, rgb)¶

Save a pixmap out.

Parameters:
  • filename (string) – The prefix for the name of the pixmap. The pixmap will be saved as “name.png” if the pixmap is RGB or Greyscale, “name.pam” otherwise.
  • rgb (int) – If non zero, the pixmap is converted to rgb (if possible) before saving.
